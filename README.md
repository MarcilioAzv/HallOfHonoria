## Sobre o Jogo

**Hall of Honoria** é um jogo de ação medieval em terceira pessoa desenvolvido utilizando a **Godot Engine**.

O jogador assume o papel do último guerreiro que permaneceu defendendo o Reino de Honoria após uma invasão bárbara devastadora. Equipado apenas com sua espada, escudo e coragem, ele deverá enfrentar os invasores e impedir a queda definitiva do reino.

O projeto foi desenvolvido com foco em sistemas de combate, inteligência artificial, movimentação 3D e ambientação medieval.

---

##  Controles

  | Tecla | Ação |
  |---------|---------|
  | **W** | Mover para frente |
  | **A** | Mover para esquerda |
  | **S** | Mover para trás |
  | **D** | Mover para direita |
  | **Espaço** | Pular |
  | **Mouse Esquerdo** | Atacar |
  | **Mouse Direito** | Defender com escudo |

---

## Funcionalidades Implementadas

- Combate corpo a corpo com espada e escudo.
- Inteligência Artificial para inimigos.
- Câmera em terceira pessoa.
- Sistema de colisões.
- Ambiente medieval com iluminação dinâmica.

## Assets Utilizados

Os personagens e alguns recursos visuais utilizados no projeto foram obtidos através do pacote:

**KayKit Adventurers**

https://kaylousberg.itch.io/kaykit-adventurers

Todos os créditos dos modelos 3D pertencem ao autor original do pacote.

### Criação do Mapa

Diferentemente dos personagens, todo o cenário principal de **Hall of Honoria** foi desenvolvido manualmente pela equipe utilizando os recursos nativos da **Godot Engine**.
